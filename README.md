# 5-practika
Смартфонды басқару жүйесі – Батарея деңгейі мен жад көлемін тексеріп басқаратын, шектеулер енгізілген телефон класын құрыңыз.

Смарт үй жүйесі – Үй температурасы мен қауіпсіздік статусын жабық атрибуттар арқылы басқарып, температура немесе қауіпсіздік шектен асса автоматты ескерту немесе сигнал шығаратын логика енгізіңіз.
